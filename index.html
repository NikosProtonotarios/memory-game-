Press A Key to Start

How to Play The Game


1. Start the Game: Press any key on your keyboard to begin the game.

2. Watch the Pattern: The game will flash a color, starting with one color in Level 1. Remember the sequence as the game progresses and adds more colors.

3. Follow the Sequence: After the pattern is displayed, it's your turn! Click the buttons in the same order the game showed. If you follow the correct pattern, you'll advance to the next level.

4. Increase the Challenge: With each new level, the game adds another color to the sequence. Keep repeating the sequence correctly to keep going!

5. Game Over: If you press the wrong button, the game will end, and you'll have to start over. Don't worry! You can always restart by pressing a key again.
